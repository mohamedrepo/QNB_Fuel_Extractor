# QNB Fuel Extractor PRO

Chrome extension to extract and analyze QNB fuel transactions.

## Features

- Export to XLSX (real Excel format)
- Date format: dd/mm/yyyy hh:mm:ss AM/PM
- Auto filters
- Live dashboard (Chart.js)
- Chrome storage sync

## Usage

1. Go to chrome://extensions
2. Enable Developer Mode
3. Load unpacked → select project folder

## Dashboard

Click "Open Dashboard" from extension popup.
