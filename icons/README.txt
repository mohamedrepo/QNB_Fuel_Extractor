# Icons for QNB Fuel Card Extractor Extension

You need to add icon files in this folder:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these using any image editor or online icon generator.
Suggested icon: A fuel pump or credit card icon in blue (#1d2552) color.

Temporary workaround: You can use any PNG images renamed to these filenames,
or the extension will work without icons (just with a default icon).
